Exercise tracker

Create a New User

POST /api/users

Add exercises

POST /api/users/:_id/exercises

API Extra Features

  1. List All User Accounts

  2. Retrieve a full exercise log of a user

  3. Retrieve a part of the log of any user

    • GET /api/users/:_id/logs?[&from][&to][&limit]

    • [ ] = optional

    • from, to = dates (yyyy-mm-dd); limit = number
